# videogames

Aplicacion examen,
Realizada con:

- ReactJS
- NodeJS
- ✨MongoDB✨

## Instalacion 

Videogames requiere [Node.js](https://nodejs.org/) v10+ para correr.

Variables de entorno.

```sh
export URI_MONGO="mongodb://127.0.0.1:27017"
export DB="VideoGames"
```

Instalacion de dependencias y correr.

```sh
cd videogames
npm i
npm run start
```
